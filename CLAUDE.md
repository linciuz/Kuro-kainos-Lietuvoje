# Kuro Kainos Lietuvoje
Static PWA (HTML/CSS/vanilla JS) showing official LT fuel prices from LEA (ena.lt).
Data: scripts/fetch_prices.py downloads LEA's daily Excel -> data/stations.json (no public API yet).
Updated daily by .github/workflows/update-fuel-prices.yml. App reads data/stations.json; filters by
municipality (no coords in source). Icons generated by tools/gen_icons.py. Deploy: GitHub Pages.
APK: PWABuilder or Bubblewrap against the live manifest.
